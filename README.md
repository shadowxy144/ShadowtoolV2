## 🇺🇸 English

### 📝 Description
**SHADOW TOOL V2** is an automated penetration testing suite designed for fast reconnaissance and vulnerability scanning. It integrates industry-standard tools into a seamless Bash interface.

### 🚀 Features
* **Auto-Install:** Installs Nmap, Sqlmap, and Cloudflared automatically.
* **Smart Scan:** Advanced Nmap scanning with automated logging to `sonuc.txt`.
* **SQLi Hunter:** Automated SQL injection testing with optimized parameters.
* **Tunneling:** Instant local-to-web exposure via Cloudflared.
